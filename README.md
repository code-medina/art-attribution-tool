# 🎨 Art ATTRIBUTION TOOL

Una herramienta simple para generar **templates de atribución artística** listos para copiar y usar.

## 🧠 Problema

Muchos artistas y creadores no saben cómo dar crédito correctamente cuando se inspiran en otras obras.

Esto puede generar:

* Falta de reconocimiento al autor original
* Confusión entre inspiración, referencia o copia
* Problemas en comunidades creativas

## 🎯 Solución

Art ATTRIBUTION permite generar textos de atribución de forma rápida y clara.

El usuario completa algunos campos y obtiene un template listo para copiar, como por ejemplo:

* "Inspirado en [Artista]"
* "Referencia: [Obra] de [Autor]"
* "Homenaje a [Artista]"

## 👥 Público objetivo

* Artistas digitales
* Ilustradores
* Diseñadores
* Estudiantes de arte
* Creadores de contenido

## ⚙️ Funcionalidades

* Generación de templates de atribución
* Personalización (autor, obra, link)
* Botón de copiar al portapapeles
* Feedback visual al copiar
* Modo claro / oscuro

## 🚫 Limitaciones

Este proyecto **NO**:

* Determina si una obra es plagio
* Ofrece validación legal
* Analiza contenido o imágenes

## 🚀 Cómo usar

1. Ingresá el nombre del artista
2. (Opcional) Agregá la obra o link
3. Elegí el tipo de atribución
4. Copiá el texto generado

## 🛠️ Instalación

```bash
git clone github 
cd carpeta
```

Abrí `index.html` en tu navegador.

## 🌗 Modo oscuro

El usuario puede alternar entre modo claro y oscuro.
La preferencia se guarda en `localStorage`.



## 💡 Motivación

Este proyecto nace de una necesidad real observada en comunidades de artistas, donde muchos usuarios no sabían cómo citar correctamente obras o referencias.
